Pattern Classification System

SYNTIN automatically categorizes discovered patterns to determine which specialized agents should be deployed:

  • Anomaly Patterns: Unusual data relationships requiring investigation
  • Opportunity Patterns: Potential for optimization or value creation
  • Emergent Patterns: Newly forming relationships with increasing significance
  • Causal Patterns: Cause-effect relationships that can be leveraged
  • Risk Patterns: Potential issues requiring preventive action

Each pattern is evaluated on multiple dimensions including gravity (significance), confidence, novelty, actionability, and time sensitivity.
